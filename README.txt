锦度电商工具箱整合版

打开方式：
1. 解压整个文件夹，保持 index.html 与 pages 文件夹在同一层级。
2. 双击 index.html。
3. 在首页点击：拼多多工具/单品利润表、京东工具/周报统计、淘宝工具/周报统计。

说明：
- 首页到子页面全部使用 ./pages/... 相对路径。
- 子页面通过 iframe 加载，CSS/JS/文件上传/导出逻辑互相隔离。
- 原子页面依赖的在线表格解析库保留为原始 CDN 地址，以保证 Excel/CSV 解析逻辑不被破坏。

本次追加：

- 产品工具 / 恶意退换货查重 -> ./pages/malicious_return_dedup.html
- 产品工具 / 查重2.0 -> ./pages/dedup_2_0.html

说明：主入口 index.html 使用相对路径加载 pages 目录中的子页面，并通过 iframe 隔离各工具页面的 CSS、JavaScript、上传和导出逻辑。
